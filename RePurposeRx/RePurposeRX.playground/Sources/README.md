# DeepSeekApp

## Overview
DeepSeekApp is an AI-powered chat application that helps users interact with a sophisticated language model for various tasks.

## Features
- Interactive chat interface
- Sidebar navigation
- Settings management
- Real-time responses

## Technologies Used
- SwiftUI
- Async/Await
- REST APIs
- JSON parsing
- Ollama (DeepSeek R1)

## Installation
1. Open in Swift Playgrounds
2. Run server.py
3. Start chatting!
